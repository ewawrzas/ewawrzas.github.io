Recent Projects

DESSRT
Eat socially

Based on Untappd, DESSRT is a fullstack web application that allows dessert lovers to share their favorite sweets! I used Ruby on Rails and PostgreSQL to create the backend, and React-Redux to build the frontend of this delicious social networking app. Enjoy!

Slugs & Dinos

Slugs & Dinos is a browser based interactive game inspired by Bejeweled. Built with JavaScript and jQuery, this game allows you to manipulate DOM elements via drag and drop event handlers. It also features icons I customized using Adobe Photoshop.

Glo

Glo, a featherweight JavaScript library inspired by jQuery, simplifies interaction with the DOM.

Why software engineering

After graduating from Williams College and gaining six years of professional experience in education, fine arts, and translation, I decided to channel my passion for problem solving and design into a career as a software engineer.
I use my training in the arts and master's degree in special education as sources of inspiration when building software. I love to transform abstract ideas into visually appealing products that are fun to use and eaisly accessible.
When I'm not writing code you can find me cycling in upstate NY, checking out new restaurants in the city, or reading the latest Stephen King novel.
